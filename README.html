JD Boundary Uploader

JD Boundary Uploader is a QGIS plugin designed to streamline importing and preparing boundary shapefiles for the John Deere Operations Center. It automatically extracts and reprojects shapefiles from FADQ ZIP archives, provides an interactive interface to update attributes (Client, Farm, Field), and supports grouping and merging of fields. Finally, it generates a packaged ZIP archive containing the shapefile and a metadata JSON file compliant with JD requirements.

Features

  • Automated Extraction & Reprojection: Import shapefiles from ZIP archives and reproject them to WGS84.
  • Interactive Editing: Easily update attributes such as Client Name, Farm Name, and Field Name.
  • Grouping & Merging: Group and merge fields for streamlined data management.
  • Export Package: Generate a ZIP archive containing the shapefile and a metadata JSON file ready for upload.

Installation

  1. Clone or download this repository.
  2. Copy the jd_boundary_uploader folder to your QGIS plugins directory.
  3. Restart QGIS and enable the plugin via the Plugin Manager.

Usage

  1. Launch the plugin from the toolbar.
  2. Select a FADQ ZIP file containing the boundary shapefile.
  3. Edit attributes as needed using the interactive interface.
  4. Use grouping and merging functions if required.
  5. Export the final ZIP archive, including the shapefile and metadata JSON file, ready for the John Deere Operations Center.
